What is this?

This page contains all of the photographs I've taken of the Xrite ColorChecker Passport with various camera models, a project I've undertaken for the benefit of darktable, an open source image editing program.

To learn what these images are useful for, please consult Pascal de Bruijn's excellent article on creating camera profiles for darktable.
